# Descripción del Proyecto

Este proyecto consiste en el desarrollo de un sistema web de gestión empresarial para un SPA, el cual permitirá administrar reservas, servicios, salas y usuarios del establecimiento.

El sistema tiene como objetivo facilitar la gestión de turnos y evitar conflictos en la asignación de especialistas o cabinas.

El desarrollo se realiza de manera colaborativa utilizando control de versiones mediante Git y un repositorio remoto en GitHub.

# Tecnologías Utilizadas
Componente	            Tecnología
Backend	                Laravel (PHP 8.2.12)
Frontend	            Vue.js
Base de Datos	        PostgreSQL
Control de Versiones	Git
API	                    REST
Autenticación	        JWT
Servidor	            Apache / Nginx

Arquitectura del Sistema

El sistema sigue una arquitectura SPA (Single Page Application) con separación entre frontend y backend.
